## Route Maps

As well as running this app locally you can visit http://routemaps.surge.sh/

Install npm packages

    npm install

To run app locally

    npm start

Code style test and unit tests

    npm test
    
Build app

    npm run build
    
Deploy app to surge.sh

    npm run deploy
